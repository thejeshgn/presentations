Solution

Communication

Solution

  1. Allow creators to communicate to middleman
  2. Middleman will publish the data and provide searh engine
  3. Users will use search engine

Simple

Not really

  • Format for communicating metadata - OpenData.json - Similar to Google sitemap, but json
  • Way to receive the that - Standard Ping (used by Google, Wordpress etc)
  • Many things to Research
  • Many things to Build
