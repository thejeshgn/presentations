Lab 9 - Collaborating

Created Saturday 29 November 2014 - Lab Home
A big part of working on data stories is collaboration. Most of the collaboration happens online. Since our focus is on data we will look at how do we collaboratively work on data.

Sample Exercise Questions

  • Collaboratively edit the 2010 election results
  • Fix the data errors in a spreadsheet as a group
  • Write the story as a group, together

Tools

Google Docs/Drive

  • Create and edit together
  • Can be shared with a limited set of people or the whole world
  • Control the access by individual
  • Can edit spreadsheets and docs

Etherpad based

  • Easy, real-time collaborative document editing
  • Text document
  • Open to public
  • Import/Export, Embed possible
  • Mozilla Pad, Pirate Pad

Hackpad

  • Easy, real-time collaborative document editing
  • Text document
  • Import/Export, Embed possible
  • Can sync to dropbox

Other tools

  • GitHub and BitBucket can be used for both code, data collaboration
